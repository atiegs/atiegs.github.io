Chapter 3

  1. Keep track of team members: 2001 --> Mitch Kapor, Morgen Sagen, Al Cho, Andy Hertzfeld
    2. In 2001 they began meeting to decide on what software they were to create.

  2. When introducing a new technology or design, why did Frederick Brooks advise “plan to throw one away?”
    3. He advised this since it is very uncommon to get things right on the first attempt.

  3. 2002, Katie Parlante joined Chandler. When she first joined what did she do?
    4. When Parlante joined, she discovered ways PIM could make meetings and phone calls easier.

  4. John Anderson joined Chandler as “systems architect.” Briefly, what did that mean?
    5. Anderson decided on the look of menus and dialogue boxes, the programming language that would be used, and the types of storage to be implemented.

  5. What is a “core” dump? Why the use of the word core?
    6. A computer’s active memory is its “core.” A core dump occurs when a major error inhibits the system and the computer displays an error file that describes what went wrong.

  6. How did Erik Sink, speaking of abstractions, describe what programmers do?
    7. Sink describes programming as building large amounts of abstractions which are then layered on top someone else’s abstractions.

  7. Rather than writing program statements in binary code, 110101110 1001101111, programmers developed a shorthand language called what?
    8. Fortran (Formula Translating System) was the first widely accepted method of shorthand programming language.

  8. Adding layers of abstraction, new programming languages were created: Lisp, Cobol, Algol, Basic. Fortran was the first widely used. What kind of program converted Fortran to binary?
    9. An “uberprogram” called a compiler translates Fortran to binary.

  9. What are the implications behind: “...there is no Moore’s Law for software. Chips may double in capacity every year or two; our brains don’t.”?
    10. It is saying to not overuse one’s time when it comes to optimizing code to fit the smallest possible amount storage. Let the computer do the hard work.

  10. The language selected for Chandler was Python. What does it mean that Python was an interpreted language?
    11. The source code being created is converted to binary while it is being written, as opposed to before the program is run. This has positives in that it is more dexterous and negatives in that is a slower process.

  11. The announcement of Chandler occurred when?
    12. It was announced on October 20th, 2002 by the Mercury News.
