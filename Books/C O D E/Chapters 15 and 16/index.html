CODE Chapters 15 and 16

Chapter 15

  1. Define Byte.
    2. A byte is another term for 8 bits of data.

  2. What is the range of an unsigned byte? A signed two's complement byte?
    3. Unsigned bytes contain positive integers ranging from 0-255 (256 in total). A two’s complement bytes show negative integers ranging from -128-127.

  3. What does the X in the following binary number represent 1010X10101?
    4. X could represent 2^6 or 64.

  4. What is the base 16 number system called?
    5. The base 16 number system is referred to as the hexadecimal system.

  5. Why is base 16 a good choice for representing binary numbers?
    6. Base 16 works because it is a power of 2 (as well as 4). This allows for an even division of bits.

  6. What base 16 number is 101011000011? What is an easy way to determine this without using a calculator?
    7. 16 is a power of four so 101011000011 can be broken down into groups of 4. 1010, 1100, and 0011. By observing the chart in this chapter, one can find the following conversions: 1010=A, 1100=C, 0011=3. Therefore the hexadecimal value is AC3.

  7. What does the X represent in the following base 16 number D4AXF0?
    8. X could represent 16^2 or 256.

  8. The book describes how to convert decimal to base 2 or base 16. Research and describe a technique that allows conversion of decimal to any base (2-36).
    9. Here is a method I found of converting a decimal to any base:
    N ÷ B = D, R -> D = N, R = M
    N is the decimal number.
    B is the base number.
    D is the result.
    R is the remainder.
    M is the completed conversion number.
    The process repeats itself until the decimal is equal to 0.
    As an example, the process to convert the decimal 7 to binary would look like this:
    7 ÷ 2 = 3, 1 -> 3=3 1=1 -> 1
    3 ÷ 2 = 1, 1 -> 3=3 1=1 -> 11
    1 ÷ 2 = 0, 1 -> 0=0 1=1 -> 111

Chapter 16

  1. What do the prefix names kilo, mega, giga, tera, and peta represent in bytes and what were the Greek origins of the words?
    2. A kilobyte is 1024 bytes. In Greek, kilo comes from the term khilioi which means 1000. However, since binary is based on the power of 2, a kilobyte cannot equal 1000 and therefore must equal the closest base 2 number: 1024.
    Mega comes from the Greek term megas which means "great". A megabyte is 1,048,576 bytes.
    Giga comes from the Greek gigas, meaning giant. A gigabyte is 1024 megabytes.
    Tera comes from the Greek teras, meaning monster. A terabyte is 1,099,511,627,776 bytes.
    Peta comes from Greek meaning 5, denoting the 5th power in base 1000 and the 15th in base 10. A petabyte is 1,125,899,906,842,624 bytes which is approximately a quadrillion.

  2. At the end of Chapter 16 the author reminds the reader about something that is "very important," what is this?
    3. This very important thing is regarding memory. RAM (random-access memory), sometimes referred to as volatile memory, will disappear without a power source. This means that everything in a RAM chip will be erased if there is no power.
